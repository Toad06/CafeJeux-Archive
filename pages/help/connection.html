help?section=connection Problèmes de connexion

Afin de faire fonctionner les jeux, les chats et d'autres fonctionnalités similaires, votre ordinateur doit permettre de se connecter à un serveur spécifique CaféJeux.

Vous devez disposer des éléments suivants :

  • un navigateur supportant JavaScript : par exemple, Internet Explorer 6 et 7 ou Firefox sont recommandés.
  • le lecteur Flash 8 ou plus.

Pare-feu internet

Si vous avez un pare-feu internet installé sur votre ordinateur, vous devez ouvrir le port sortant TCP 1024. Pour savoir comment faire cela, veuillez consulter la documentation de votre pare-feu.

Si votre configuration est bonne, l'icône de tasse en haut à droite indiquera ONLINE.
