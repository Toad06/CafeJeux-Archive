Sucres blancs

Chaque jour, vous recevez 3 sucres BLANCS gratuitement ! Les sucres blancs vous permettent de jouer à tous les jeux du Café. Toutefois, si vous ne les utilisez pas, ils seront perdus dès le lendemain.

Sucres roux

Vous pouvez obtenir des sucres ROUX supplémentaires depuis la section Plus de parties. Ces sucres vous permettent de :

  • jouer à tous les jeux du Café.
  • créer votre propre table de jeu.
  • passer des annonces pour vos tables.
  • offrir des sucres à d'autres joueurs.
  • Important : Les sucres roux peuvent être utilisés quand vous le désirez, ils ne seront pas perdus dès le lendemain !
