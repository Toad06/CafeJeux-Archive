Niveau du jeu: moyen

But du jeu

Massacrez les cosmos ennemis grâce a un arsenal d'armes surpuissantes. Note : ce jeu comporte des bugs qui peuvent survenir durant la partie.

Déroulement du jeu

Rapprochez votre souris du cosmo actif pour faire apparaitre le menu de selection. Choisissez votre mode (une arme, ou les fleches pour se déplacer) en cliquant sur l'icone correspondante.

Mode mouvement :

Cliquez autour de votre Cosmo grace a la souris pour le déplacer.

Mode Tir :

Choisissez l'angle de tir, puis tirez en cliquant avec votre souris.
