help?section=group Les tables de jeu

Les tables de jeu sont des groupes de joueurs qui vous permettent de rencontrer d'autres personnes et de jouer entre amis.

Rejoindre une table

Vous pouvez rejoindre une table à partir du lien Tables de Jeu dans le menu de gauche. Sur cette page, vous pourrez alors voir les dernières annonces postées par les créateurs de tables, ainsi que la liste des dernières tables créées.

Vous pouvez par simple clic accéder à la présentation de la table. Si le nom, la description et les jeux auxquels cette table joue vous correspondent, vous pouvez alors faire une demande pour rejoindre la table.

Une fois votre demande enregistrée, il vous faudra patienter en attendant que le créateur de la table accepte votre demande : vous ferez alors partie de cette table et elle s'affichera dans le menu de gauche.

Vous pouvez être membre de 8 tables simultanément au maximum.

Les membres

Chaque table peut réunir jusqu'à 60 membres.

Par défaut, seul le créateur de la table peut accepter, refuser ou supprimer un membre. Il peut aussi donner ces droits à d'autres membres s'il souhaite déléguer un certain nombre de tâches.

Le chat et le forum

Le chat de chaque table est le lieu où se retrouvent les membres de la table pour discuter entre eux. Vous pouvez y voir tous les membres de la table qui sont actuellement connectés au Café Jeux et les défier pour marquer des points au classement de la table (voir ci-dessous).

Chaque table possède aussi son propre forum privé, réservé aux membres, qui permet de discuter de l'actualité de la table ou de tout autre sujet autorisé par le créateur de la table.

Le classement

Tous les membres de la table participent au classement de la table.

Le classement est géré par le créateur de la table qui choisit les jeux et la durée du classement. Vous pouvez défier un autre joueur de la table dans le cadre d'un Match de Table et ainsi progresser dans le classement de la table.

En fonction de la réputation des membres de la table, être bien classé sur cette table pourra être un véritable défi !

Créer une table

Si vous désirez créer et gérer votre propre table de jeu, vous pouvez aussi le faire à partir de cette page.

Créer votre propre table vous permettra de gérer l'acceptation des membres de cette table, de choisir sur quels jeux les joueurs de cette table vont s'affronter et de personnaliser l'espace de la table en fonction de vos goûts.
